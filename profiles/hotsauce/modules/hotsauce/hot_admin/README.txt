HotAdmin!
==============
This is the administrative heart of Hotsauce. It provides integration with popular UX/UI
enhancement modules and exposes a variety of end user settings for Hotsauce.

Key Features
* Customized Content Editing Experience
* Custom Style for the Media Browser
* Creates HotAdmin!istrative Section (admin/Hotsauce) for Hotsauce Configuration
* Provides Pane Library, Layout Library, and Page Library Administrative Pages
* Packages Several Different Administrative Toolbar Options (Admin Menu, Admin, Navbar, Toolbar)
